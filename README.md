## 📋 О программе

Графическое приложение для автоматической генерации документов курсовых работ:
- 📄 **Титульные листы**
- 📝 **Задания**
- ✍️ **Отзывы руководителя**

Программа читает данные из Excel и подставляет их в шаблоны Word.

## ✨ Возможности

- ✅ Загрузка данных из Excel файла
- ✅ Выборочная генерация для конкретных студентов
- ✅ Фильтрация по группе и руководителю
- ✅ Поиск по ФИО
- ✅ Прогресс-бар и детальная статистика
- ✅ Горячие клавиши (Ctrl+A, Ctrl+D, Ctrl+G и др.)
- ✅ Автоматическое сохранение настроек
- ✅ Удобная структура папок по группам

## 🚀 Как установить и запустить

### 1. Клонировать репозиторий
```bash
git clone https://github.com/MrPacmann/course-work-generator.git
cd course-work-generator
```

### 2. Установить зависимости
```bash
pip install pandas python-docx openpyxl
```

### 3. Запустить программу
```bash
python main.py
```

## 📊 Формат Excel файла

Необходимые колонки в таблице:

| Колонка | Описание |
|---------|----------|
| Студент ФИО | Полное ФИО студента |
| Студент инициалы | Фамилия с инициалами |
| Группа | Номер учебной группы |
| РУКОВОДИТЕЛЬ | ФИО руководителя |
| Тема курсовой работы | Название работы |
| ДАТА1...ДАТА9 | Даты (опционально) |

## ⌨️ Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+A` | Выбрать всех студентов |
| `Ctrl+D` | Снять выбор |
| `Ctrl+G` | Начать генерацию |
| `Ctrl+O` | Открыть папку с документами |
| `Ctrl+F` | Фокус на поиск |
| `Ctrl+S` | Открыть настройки |
| `F1` | Помощь |

## 🎯 Плейсхолдеры в шаблонах

В шаблонах Word можно использовать:

- `__СТУДЕНТ__` — полное ФИО студента
- `__ИНИЦИАЛЫСТ__` — фамилия с инициалами
- `__ГРУППА__` — номер группы
- `__РУКОВОДИТЕЛЬ__` — ФИО руководителя
- `__ТЕМА__` — тема работы
- `__ДОЛЖНОСТЬ__` — должность руководителя
- `__Название дисциплины__` — название предмета
- `__ДАТА1__` ... `__ДАТА9__` — даты
